команды

Консольные команды

Вводятся в чат прямо в игре

/sw [ID]
Открыть интерфейс
Открывает окно выдачи розыска / штрафа для указанного игрока по его ID
/swlog
Открыть лог
Открывает HTML файл с историей всех выданных розысков и штрафов
/swrec
Папка с записями
Быстро открывает папку с видеозаписями через Проводник Windows
установка

Как установить

Следуй шагам ниже — займёт меньше минуты

1
Скачать скрипт
Скачай файл fixedwanted.lua по кнопке ниже
2
Поместить в папку MoonLoader
Скопируй файл в GTA San Andreas\moonloader\
3
Настроить пути в скрипте
Открой .lua файл и укажи свои пути к папке записей и файлу лога в секции -- НАСТРОЙКИ
4
Запустить игру
Зайди в игру — скрипт загрузится автоматически. В чате появится [FWanted] Скрипт загружен!
